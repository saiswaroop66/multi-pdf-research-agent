# 📚 Multi-PDF Research Assistant

An AI-powered Multi-PDF Research Assistant built using Streamlit, LangChain, ChromaDB, Sentence Transformers, and Llama 3.2.

## 🚀 Features

- Upload multiple PDF documents
- Extract and process text from PDFs
- Chunk large documents
- Generate embeddings
- Store embeddings in ChromaDB
- Retrieve relevant information using semantic search
- Ask questions across multiple PDFs
- Answer questions using Llama 3.2

## 🛠️ Technologies Used

- Python
- Streamlit
- LangChain
- ChromaDB
- Sentence Transformers
- Ollama
- Llama 3.2
- PyPDF

## ⚙️ Installation

### Clone Repository

```bash
git clone <your-repository-link>
cd multi-pdf-research-assistant
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Install Llama 3.2

```bash
ollama pull llama3.2
```

### Run Application

```bash
streamlit run app.py
```

## 📖 Usage

1. Upload one or more PDF files.
2. Wait for indexing to complete.
3. Ask questions related to the uploaded PDFs.
4. Get AI-generated answers based on document content.
